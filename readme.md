ECMAScript6 Tutorial
------

# Required

* babel
* harp

```sh
$ npm install --global babel
$ npm install --global harp
```

# Run

```sh
$ harp server
```

# Compile

* Compile Directories

```sh
$ babel src --watch --out-dir build
```

* Compile Single File

```sh
$ babel script.js --watch --out-file script-compiled.js
```
